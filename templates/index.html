What is this?

PELD-Fleet is the web component for the fleet networking feature of PyEveLiveDPS (PELD).
It provides a webpage for FCs to do fleet management and read output of combat stats from all of their fleet members running PELD.
This allows FCs to see stats like fleet location, and whom/what their fleet are attacking in realtime. This helps FCs decide on who to primary, to ensure line members are engaging the correct targets, make sure logi chains are correct, etc.


How do I use it?

See the help page for some tutorials.
An FC can use PELD-Fleet without any fleet members running PELD itself, but you will only get combat data from fleet members running PELD.
The FC will need the Fleet Boss role to use this site. This is due to limitations set by CCP.
